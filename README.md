# Car-SQL-Data-Analysis
## About This Project

This project was completed as part of my Data Analytics Internship. The objective was to use SQL queries to analyze a car dataset and extract meaningful insights from the data.

## What I Did

* Imported the car dataset into a SQL database
* Used SQL queries to explore the data
* Applied filtering, sorting, and grouping techniques
* Calculated statistics such as count and average
* Identified trends and patterns in the dataset

## SQL Operations Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* COUNT()
* AVG()

## Tools Used

* Python
* SQLite
* Pandas
* VS Code

## Key Findings

* Calculated the total number of cars in the dataset
* Found the average car price
* Identified cars manufactured after 2020
* Analyzed the number of cars available for each brand
* Compared average prices across different brands

## Conclusion

This project helped me understand how SQL can be used to retrieve, filter, and analyze data efficiently. It improved my understanding of database querying and data analysis techniques.
